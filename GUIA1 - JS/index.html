Preguntas Complementarias

Pregunta 1 - ¿Qué es JavaScript no intrusivo? - Ejemplificación

JavaScript no intrusivo hace referencia a la acción de escribir código en un archivo exclusivo solo para JS, en el cual se conecta mediante una etiqueta html "script" haciendo referencia al código que está escrito para la funcionalidad de la página tomando como variables los objetos html.

Pregunta 2 - ¿Cuál es la diferencia entre términos null, undefined

Null es un valor de significado especial de "no valor", null es además un objeto especial porque el typeof() de null retorna un objeto. De otra manera, undefined significa que la variable que no ha sido declarada, o no ha sido con un valor dado.
